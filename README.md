# Included Callouts
- Check Well Being
- Fake ID

- Suspicious Vehicle

# How to install
- Open your ERS resource in your Server Resource Directory
- Go to Callouts > Plugins
- Drag and drop the .lua files provided in the download into the Plugins folder for each callout.

# How to add/change Callout Locations
- Open the .lua file (Example: bz_fake_id_callout.lua)
- Line 19 of bz_fake_id_callout.lua is the start of the spawn locations. Ensure the numbers in the brackets continue on as you add locations.
RECOMENDED >40 locations for all callouts.
